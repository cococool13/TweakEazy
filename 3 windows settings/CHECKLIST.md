# Windows 11 Settings Checklist for Gaming

Open each shortcut in this folder to jump directly to the right Settings page.
Check off each item as you go.

---

## Display (display-settings.url)
- [ ] Set your monitor to its **highest refresh rate** (144Hz, 165Hz, 240Hz, etc.)
- [ ] Enable **HDR** if your monitor supports it
- [ ] Enable **Hardware-accelerated GPU scheduling (HAGS)** — can reduce input lag

## Notifications (notifications-settings.url)
- [ ] Turn off **notifications from apps** you don't need
- [ ] Enable **Do Not Disturb** during gaming sessions (or set a schedule)

## Storage (storage-settings.url)
- [ ] Turn on **Storage Sense** to auto-clean temp files
- [ ] Set it to run **every month**

## Mouse (mouse-settings.url)
- [ ] **DISABLE** "Enhance pointer precision" (this is mouse acceleration — bad for gaming)
- [ ] Set pointer speed to **6/11** (the "no acceleration" sweet spot)
- [ ] If you have a gaming mouse, use its software to set DPI instead

## Startup Apps (startup-settings.url)
- [ ] **Disable** everything you don't need at startup
- [ ] Common safe-to-disable: Discord, Spotify, OneDrive, Teams, Cortana
- [ ] Keep: GPU drivers (NVIDIA/AMD), audio drivers, security software

## Installed Apps (apps-settings.url)
- [ ] Uninstall bloatware: Clipchamp, News, Weather, Maps, People, Solitaire
- [ ] Uninstall apps you never use
- [ ] Don't uninstall: Microsoft Store, Windows Terminal, .NET frameworks

## Game Mode (gaming-settings.url)
- [ ] **Enable** Game Mode — prioritizes GPU/CPU for games
- [ ] This is ON by default in Win11 but verify it

## Graphics Settings (graphics-settings.url)
- [ ] Set your games to use **High performance** GPU (if you have integrated + discrete)
- [ ] Add your game .exe files manually if they don't appear in the list

## Visual Effects (visual-effects-settings.url)
- [ ] Select **"Adjust for best performance"**
- [ ] Then re-enable **"Smooth edges of screen fonts"** (text looks terrible without it)

## Privacy & Telemetry (privacy-settings.url)
- [ ] Disable **Advertising ID**
- [ ] Disable **Speech recognition** (if not using)
- [ ] Disable **Inking & typing personalization**
- [ ] Set **Diagnostic data** to "Required" (minimum)

## Focus / Do Not Disturb (focus-settings.url)
- [ ] Set up a **Focus session** profile for gaming
- [ ] This silences all notifications and popups while gaming

---

## Quick Summary

| Setting | Change | Why |
|---------|--------|-----|
| Refresh rate | Max available | Smoother gameplay |
| HAGS | On | Lower input lag |
| Enhance pointer precision | OFF | Removes mouse acceleration |
| Startup apps | Minimize | Frees RAM and CPU |
| Game Mode | On | Prioritizes game processes |
| Visual Effects | Best performance | Frees GPU resources |
| Do Not Disturb | On (gaming) | No popup interruptions |
